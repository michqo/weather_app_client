# Weather app client

**Client for [weather_app](https://github.com/MichalUSER/weather_app) built with micropython**

## Requirements

- ESP32 development board
- Python 3.8.0 (install with [pyenv](https://github.com/pyenv/pyenv))
- Installed ampy with pip
